[`@pagopa/azure-tracing`](https://github.com/pagopa/dx/tree/main/packages/azure-tracing)
is the DX package that integrates Azure Monitor and Application Insights with
OpenTelemetry in Node.js applications, with specific support for Azure
Functions.

## Use cases

- Enabling distributed tracing in Azure Functions and Node.js services
- Correlating logs, traces, and dependencies in Application Insights
- Standardizing telemetry setup across DX-based Azure workloads
- Adopting managed identity authentication for Azure Monitor telemetry
