Add Azure Managed Redis support via new `managed_redis` input variable.

The `managed_redis` input accepts a plain list of AMR resource IDs. The module
validates each ID using `provider::azurerm::parse_resource_id` and creates an
access policy assignment with the built-in `default` policy.

Example:
```hcl
managed_redis = [
  azurerm_managed_redis.cache.id,
]
```

Legacy `redis` input is unchanged (backward compatible).
