# Experiments: Terraform Skill Evaluation

This folder contains a minimal harness to evaluate agent approaches for
generating Terraform infra using PagoPA DX best practices.

Purpose

- Provide repeatable scripts to run generation approaches (mcp, skill-inline,
  skill-rag, local docs, subagent)
- Collect artifacts and basic metrics

How to use

1. Run `scripts/run_experiment.sh --approach skill-rag` (see script for args)
2. Results will be stored in `results/<approach>/`

Notes

- Files created here are meant to be used locally and are not committed by the
  harness by default.

# Terraform Skill Evaluation - Experiments

This folder contains a minimal harness to evaluate different approaches for the
`terraform-dx-best-practices` skill (MCP, skill-inline, skill+RAG, local docs,
subagent pattern).

Structure

- `scripts/` - helper scripts to run experiments
- `prompts/` - canonical task prompt and checklist
- `results/` - generated outputs and logs per approach

Quick start

Run the search API check first:

```bash
# from repo root
bash experiments/scripts/test_search_api.sh "Terraform code style"
```

Run an experiment (dry-run scaffolding):

```bash
bash experiments/scripts/run_experiment.sh --approach skill-rag
```

Outputs

- `results/<approach>/` contains generated files, logs and `metrics.json`
- `results/report.md` contains a human-readable comparison (generated by scripts)
